# AnkiCards-Flutter
AnkiCardsは、フラッシュカードを作成するときに面倒なことを軽減するために開発しました。煩わしいサインアップ、広告、ロックされた機能は存在しません！
AnkiCardsは以下の機能をサポートします。

# 機能(実装ずみ)
- カードの作成、編集、削除
- Material3のUI
- 広告なし
- 継続したアップデート

# 間もなく登場
- 出題機能
- テーマの自動切り替え
- Gemini搭載(構成、カードの生成)
- タグによる分類
- 高度な復習機能(リマインド)
- ダッシュボード
- iPadOSに最適化されたUI
