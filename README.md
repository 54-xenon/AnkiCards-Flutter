# AnkiCards Flutter

AnkiCardsは、フラッシュカード作成の「面倒」をなくすことを目指した学習アプリです。サインアップ不要、広告なし、必要な機能に絞ったシンプルな体験を提供します。

## 特徴
- カードの作成・編集・削除
- 出題（学習）機能
- 結果の表示と保存
- Geminiによるカード解説の自動生成
- Material 3 UI
- 広告なし
- 日本語環境向けのローカライズ
